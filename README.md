# Student Information Management System (C++)

A robust command-line application designed to manage student academic records. This project demonstrates my ability to work with **Custom Data Structures** and **File/Data Management** logic in C++.

### Key Features
- **Record Management:** Add, Edit, Delete, and Search student records.
- **Data Organization:** Uses `struct` to group related data (Name, ID, GPA, etc.).
- **Dynamic Listing:** Utilizes `std::vector` (or arrays) for efficient handling of student lists.
- **Input Validation:** Ensures data integrity during entry.

### Technical Concepts
- **Structures (struct):** For organizing complex student entities.
- **Looping & Logic:** Advanced switch-case menus for a seamless user experience.
- **Search Algorithms:** Implementing basic search logic by Student ID.
